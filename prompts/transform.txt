## 核心任务目标

1. **内容完整性**：不可将长文缩减为短讯。在保持逻辑清晰的前提下，必须保留原文的生动细节、感人场景、校方金句和专业术语。

2. **SEO 优化方向**：内容需突出我校的"办学实力"、"名校资源"、"学生成长"和"先进教育理念"等等高搜索价值的方向靠拢，以吸引家长和潜在生源。

3. **格式规范**：Frontmatter 必须严丝合缝，正文必须结构化（多级标题）。

4. **图片处理**：为每张图片添加描述性 alt 文本，格式为 `![图片描述](URL)`，描述需结合上下文（如：`![仲恺学子在上海交大聆听学长分享会](图片URL)`）。**注意：文中的装饰性图片已在预处理阶段由视觉AI自动移除，请保留文中所有剩余图片，不要删除任何图片。**


## 严格分类体系

**必须且仅能从以下主分类中选择：**

| 分类 | 说明 | 关键词/场景 |
|------|------|------------|
| `新闻动态` | 综合性新闻，不属于以下特定类别 | 学校重大新闻、对外交流 |
| `校园新闻` | 学生视角的校园生活、班级故事 | 班级活动、学生日常 |
| `通知公告` | 事务性内容 | "通知"、"公告"、"关于..."、"请..."、"定于..." |
| `教务动态` | 教育教学活动 | 教学、课程、考试、教研、备课、课堂 |
| `校园活动` | 各类活动 | 比赛、演出、运动会、社团、实践、研学 |
| `课程教学` | 具体课程介绍 | 课程改革、教材、教学方法 |
| `师资力量` | 教师介绍 | 教师风采、名师介绍、教师培训 |
| `办学成果` | 学校层面成果 | 成果总结、数据统计、升学率 |
| `荣誉时刻` | 获奖表彰 | 获奖、表彰、第一名、金牌、优秀 |

**分类判断优先级（按顺序判断）：**

1. 包含"通知"、"公告"、"关于..."、"请..."、"定于..."等 → **通知公告**
2. 涉及教学、课程、考试、教研、备课等 → **教务动态**
3. 涉及比赛、演出、运动会、社团、研学等 → **校园活动**
4. 涉及获奖、表彰、第一名、金牌、优秀等 → **荣誉时刻**
5. 主要介绍教师个人或团队 → **师资力量**
6. 介绍具体课程、教材改革、教学方法 → **课程教学**
7. 学生视角的校园生活、班级故事 → **校园新闻**
8. 学校层面的成果总结、数据统计 → **办学成果**
9. 其他综合性新闻 → **新闻动态**

## 正文处理规范

### 1. 导语编写
- 编写引人入胜的导语，概括文章核心内容
- 导语要有吸引力，能激发读者继续阅读的欲望

### 2. 结构化处理
- 使用小标题串联全文，确保每个板块内容充实
- 标题层级：一级标题用 `##`，二级标题用 `###`
- 删除原文的装饰性符号（如 ═══、━━━ 等）

### 3. 语言优化
- 修正口语化表达，使用书面语
- 统一标点符号（使用中文标点）
- 保持昂扬向上的文风

### 4. 图片处理
- **保留文中所有图片**，不要删除任何图片
- 装饰性图片已在预处理阶段自动移除，文中剩余图片均为主题内容图
- 为每张图片添加描述性 alt 文本
- alt 文本应结合上下文，描述图片实际内容
- 示例：`![仲恺中学2024届高考百日誓师大会现场]`

### 5. 结尾处理
- 末尾统一添加：`> 本文章来自惠州仲恺中学公众号`
